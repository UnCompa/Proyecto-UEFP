## Conociendo el entorno

Para el desarrollo web es fundamental estas tecnologias las cuales son:

- **HTML**
- **CSS**
- **JavaScript**

Con estas tres tecnologias se abren un mundo de posibilidades para el desarrollo web, a continuación se dará más detalle de cada una

Estas tres tecnologías abren un mundo de posibilidades para el desarrollo web. A continuación, se proporciona más detalle sobre cada una de ellas.

## HTML

HTML es la tecnología fundamental en el desarrollo web, ya que con ella se estructura y da forma a nuestra página web.

### Sintaxis

La sintaxis de HTML es bastante intuitiva, ya que se maneja a través de etiquetas. Esto permite estructurar nuestra web de manera clara y organizada. Aquí tienes un ejemplo:

```html
<h1>Hola soy una etiqueta de HTML</h1>
```

Con este resumen se da la importancia de HTML, para aprender mas se dedico una seccion específica para aprender las bases de este [Aprender mas](/es/docs)

## CSS

CSS permite dar estilo a nuestra web, logrando así cualquier tipo de diseño que imaginemos.

### Sintaxis

Su sintaxis se basa en seleccionar elementos HTML y aplicarles propiedades específicas para modificar su apariencia. Aquí tienes un ejemplo:

```css
/*Aqui esta el selector el cual es HTML*/
html {
  background-color: blue; 
  /*Luego esta la propiedad: "background-color", con el valor "blue"*/
}
```

## JavaScript

JavaScript es el lenguaje de programación de la web. Con él, podemos crear páginas dinámicas y añadir interactividad a nuestra web.

### Sintaxis

La sintaxis de JavaScript es única debido a su naturaleza como lenguaje de programación, ofreciendo una amplia variedad de funcionalidades. Para obtener más información, visite nuestra guía dedicada: [Dando vida a la web con JavaScript](/es/docs)

Aqui hay un pequeño ejemplo

```js
//main.js
console.log("Hola mundo")

let numero = 10
let nombre = "Juan"
```

## Conclusiones

Estas tecnologias son las fundamentales para empezar, con esto puedes conocer más sobre los requerimiento para la web como pueden ser:

- **React**
- **Git**
- **Astro JS**
- **Tailwind CSS**
- **Markdown y MDX**
