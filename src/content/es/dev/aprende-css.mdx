CSS, o Cascading Style Sheets, es el lenguaje que se utiliza para describir la presentación de documentos HTML. Con CSS, puedes controlar el diseño, la disposición y el aspecto visual de tus páginas web.

## ¿Por qué aprenderlo?

Aprender CSS te permitirá transformar páginas HTML simples en sitios web atractivos y profesionales. Con CSS, puedes definir estilos, ajustar diseños responsivos, y mejorar la experiencia de usuario.

## Recursos para aprender

CSS es amplio y detallado, por lo que se proporcionarán recursos de aprendizaje y un temario para guiar tu estudio de CSS.

## Temario

### Módulo 1: Introducción a CSS

1. **Historia y Evolución de CSS**
   - Orígenes de CSS
   - Evolución de las versiones de CSS
   - Principales mejoras de CSS3
2. **Conceptos Básicos de CSS**
   - ¿Qué es CSS?
   - Sintaxis básica y estructura de un archivo CSS
   - Métodos de inclusión de CSS: inline, interno y externo

### Módulo 2: Selectores y Propiedades

1. **Selectores Básicos**
   - Selectores de tipo, clase e id
   - Selectores de atributo
   - Selectores descendentes y de hijo directo
2. **Propiedades y Valores**
   - Propiedades básicas: color, background, font
   - Unidades de medida: px, em, rem, %, vh, vw
   - Uso de variables CSS (custom properties)

### Módulo 3: Modelo de Caja (Box Model)

1. **Entendiendo el Modelo de Caja**
   - Margen, borde, padding y contenido
   - Modelo de caja estándar vs modelo de caja alternativo (`box-sizing`)
2. **Manipulación del Box Model**
   - Ajuste de márgenes y padding
   - Control del tamaño y el borde

### Módulo 4: Diseño y Maquetación

1. **Diseño con Flexbox**
   - Conceptos básicos de Flexbox
   - Propiedades principales: `display`, `flex-direction`, `justify-content`, `align-items`
   - Alineación y distribución de elementos
2. **Diseño con Grid**
   - Conceptos básicos de Grid Layout
   - Definición de contenedores y elementos de grid
   - Propiedades de Grid: `grid-template-columns`, `grid-template-rows`, `gap`

### Módulo 5: Estilización Avanzada

1. **Pseudo-clases y Pseudo-elementos**
   - Uso de pseudo-clases: `:hover`, `:focus`, `:nth-child`, etc.
   - Uso de pseudo-elementos: `::before`, `::after`
2. **Transiciones y Animaciones**
   - Transiciones básicas: `transition-property`, `transition-duration`
   - Creación de animaciones con `@keyframes`

### Módulo 6: Responsive Design

1. **Media Queries**
   - Introducción a media queries
   - Sintaxis y uso de `@media`
   - Diseño adaptable para diferentes dispositivos
2. **Diseño Fluid y Adaptativo**
   - Uso de unidades relativas y fluidas
   - Técnicas de diseño adaptativo

### Módulo 7: Prácticas y Proyecto Final

1. **Prácticas Guiadas**
   - Ejercicios prácticos para reforzar los conceptos aprendidos
   - Desarrollo de ejemplos de diseño web con CSS
2. **Proyecto Final**
   - Descripción del proyecto
   - Requisitos y criterios de evaluación
   - Presentación y revisión del proyecto

### Módulo 8: Buenas Prácticas y Accesibilidad

1. **Buenas Prácticas en CSS**
   - Código limpio y legible
   - Uso adecuado de comentarios
   - Optimización de rendimiento
2. **Accesibilidad**
   - Importancia de la accesibilidad en la web
   - Herramientas y técnicas para crear contenido accesible
