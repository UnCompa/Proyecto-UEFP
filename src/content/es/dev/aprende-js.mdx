JavaScript es el lenguaje de programación más popular en el desarrollo web, permitiendo crear páginas dinámicas e interactivas.

## ¿Por qué aprenderlo?

Aprender JavaScript te permitirá agregar interactividad a tus páginas web, mejorar la experiencia de usuario, y desarrollar aplicaciones web completas tanto en el frontend como en el backend.

## Recursos para aprender

JavaScript es muy amplio y detallado, por lo que se proporcionarán recursos de aprendizaje y un temario para guiar tu estudio de JavaScript.

## Temario

### Módulo 1: Introducción a JavaScript

1. **Historia y Evolución de JavaScript**
   - Orígenes de JavaScript
   - Evolución y versiones de ECMAScript
   - Principales características de ES6+
2. **Conceptos Básicos de JavaScript**
   - ¿Qué es JavaScript?
   - Sintaxis básica y estructura de un script
   - Inclusión de JavaScript en HTML: inline, interno y externo

### Módulo 2: Fundamentos de Programación

1. **Variables y Tipos de Datos**
   - Declaración de variables: `var`, `let`, `const`
   - Tipos de datos primitivos: `number`, `string`, `boolean`, `null`, `undefined`
   - Tipos de datos complejos: objetos y arrays
2. **Operadores y Expresiones**
   - Operadores aritméticos, de asignación, de comparación, y lógicos
   - Expresiones y evaluación

### Módulo 3: Estructuras de Control

1. **Condicionales**
   - `if`, `else if`, `else`
   - Operador ternario
2. **Bucles**
   - `for`, `while`, `do...while`
   - Iteración sobre arrays y objetos

### Módulo 4: Funciones

1. **Declaración y Ejecución de Funciones**
   - Funciones declarativas y expresivas
   - Parámetros y argumentos
   - Funciones flecha (arrow functions)
2. **Ámbito y Closures**
   - Ámbito de variables (scope)
   - Closures y su utilidad

### Módulo 5: Objetos y Arrays

1. **Manipulación de Objetos**
   - Creación y modificación de objetos
   - Propiedades y métodos
   - Destructuración de objetos
2. **Manipulación de Arrays**
   - Creación y modificación de arrays
   - Métodos útiles: `push`, `pop`, `shift`, `unshift`, `map`, `filter`, `reduce`
   - Destructuración de arrays

### Módulo 6: Programación Asíncrona

1. **Callbacks**
   - Concepto de callbacks
   - Funciones de orden superior
2. **Promesas**
   - Introducción a las promesas
   - Métodos: `then`, `catch`, `finally`
3. **Async/Await**
   - Uso de `async` y `await`
   - Manejo de errores en código asíncrono

### Módulo 7: Manipulación del DOM

1. **Selección de Elementos del DOM**
   - Métodos de selección: `getElementById`, `querySelector`, `querySelectorAll`
   - Manipulación de clases y estilos
2. **Eventos**
   - Manejo de eventos: `addEventListener`
   - Eventos comunes: `click`, `mouseover`, `submit`

### Módulo 8: Buenas Prácticas y Proyecto Final

1. **Buenas Prácticas en JavaScript**
   - Código limpio y legible
   - Uso adecuado de comentarios
   - Optimización de rendimiento
2. **Proyecto Final**
   - Descripción del proyecto
   - Requisitos y criterios de evaluación
   - Presentación y revisión del proyecto
