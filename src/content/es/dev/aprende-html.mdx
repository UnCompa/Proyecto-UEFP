El lenguaje de la web, así llamado debido a que con este lenguaje se crearon las primeras páginas web, es la base de cualquier proyecto web.

## ¿Por qué aprenderlo?

Sin duda, con este lenguaje podrás crear tus primeras páginas web. Aunque no muy llamativas al inicio, ya que se debe usar CSS para estilizar, HTML es fundamental para empezar.

## Recursos para aprender

Debido a que HTML es muy extenso y complejo de explicar en su totalidad, se proporcionarán recursos de aprendizaje y un temario para tener una idea de qué puedes aprender de HTML.

## Temario

### Módulo 1: Introducción a HTML5

1. **Historia y Evolución de HTML**
   - Orígenes de HTML
   - Evolución hasta HTML5
   - Principales mejoras de HTML5 respecto a HTML4
2. **Conceptos Básicos de HTML5**
   - ¿Qué es HTML5?
   - Estructura básica de un documento HTML5
   - Etiquetas esenciales (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)

### Módulo 2: Estructura y Semántica en HTML5

1. **Elementos Semánticos**
   - Introducción a la semántica en HTML5
   - Etiquetas nuevas: `<header>`, `<footer>`, `<section>`, `<article>`, `<nav>`, `<aside>`
   - Importancia de la semántica para la accesibilidad y SEO
2. **Elementos de Texto y Formateo**
   - Etiquetas básicas de texto: `<p>`, `<h1>` a `<h6>`, `<blockquote>`, `<pre>`, `<code>`, `<mark>`
   - Etiquetas de énfasis: `<strong>`, `<em>`, `<b>`, `<i>`

### Módulo 3: Multimedia en HTML5

1. **Imágenes**
   - Insertar imágenes con `<img>`
   - Atributos importantes: `src`, `alt`, `width`, `height`
2. **Audio y Video**
   - Insertar audio con `<audio>`
   - Atributos y opciones de control
   - Insertar video con `<video>`
   - Formatos soportados y etiquetas de control

### Módulo 4: Formularios en HTML5

1. **Nuevas Etiquetas y Atributos de Formularios**
   - Campos de entrada: `<input>`, `<textarea>`, `<select>`, `<datalist>`
   - Nuevos tipos de input: `email`, `url`, `date`, `color`, `range`
   - Validación de formularios y atributos de validación: `required`, `pattern`, `min`, `max`
2. **Elementos de Interfaz de Usuario**
   - Etiqueta `<label>`
   - Grupos de opciones con `<fieldset>` y `<legend>`

### Módulo 5: APIs y Características Avanzadas

1. **Canvas**
   - Introducción al elemento `<canvas>`
   - Dibujo básico: líneas, rectángulos, arcos, y texto
2. **SVG (Scalable Vector Graphics)**
   - Introducción a SVG
   - Comparación entre SVG y Canvas
3. **Local Storage y Session Storage**
   - Diferencias entre localStorage y sessionStorage
   - Uso de localStorage y sessionStorage para almacenar datos en el navegador
4. **Geolocalización**
   - API de Geolocalización
   - Solicitar la ubicación del usuario

### Módulo 6: Prácticas y Proyecto Final

1. **Prácticas Guiadas**
   - Ejercicios prácticos para reforzar los conceptos aprendidos
   - Desarrollo de pequeñas aplicaciones web utilizando HTML5
2. **Proyecto Final**
   - Descripción del proyecto
   - Requisitos y criterios de evaluación
   - Presentación y revisión del proyecto

### Módulo 7: Buenas Prácticas y Accesibilidad

1. **Buenas Prácticas en HTML5**
   - Código limpio y legible
   - Uso adecuado de comentarios
   - Optimización de rendimiento
2. **Accesibilidad**
   - Importancia de la accesibilidad en la web
   - Herramientas y técnicas para crear contenido accesible
