# Guia para el usuario 🚀

![Hola](/public/imgs/programacion.jpeg)

Aqui la documentación para el usuario, y entender como esta compuesta la página de la Unidad Educativa Fiscal Pichincha.
Si eres desarrollador visita nuestra documentación para seguir mejorando y manteniendo esta página. [Click aqui](/es/dev) para mas información.

## Temas
- Composición de la web
- Aspectos destacados
- Personal involucrado

### Composición de la web

La web esta compuesta por varias seccion, actualmente en la V1.0 se escuentran las siguientes secciones:

- Pagina de inicio
- Pagina sobre la historia de la institución
- Pagina para exponer los niveles de educación que se oferta
- Pagina con contacto directo y datos importantes

### Aspectos destacados

La pagina cuenta con información de calidad para entender como funciona la web, además de ofreces un avistamiento a la **Institucion**
