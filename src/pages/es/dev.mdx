# Guía para desarrolladores

¡Bienvenido al equipo de desarrollo de la Unidad Educativa Fiscal Pichincha (UEFP)! Esta guía te proporcionará una visión general del proyecto y los recursos necesarios para contribuir al desarrollo y mantenimiento de nuestra página web.

Para obtener más detalles y colaborar en el proyecto, visita nuestro repositorio en [GitHub](https://github.com/UnCompa/Proyecto-UEFP).

## Visión general del proyecto

Nuestra página web está construida con **Astro JS**, un framework web que permite crear páginas rápidas y eficientes para mostrar contenido de manera dinámica.

![Astro Logo](https://cdn.buttercms.com/xrVbfdR5TBy4iTaY4xl7)

Para obtener más información sobre Astro JS, visita la página oficial de [Astro JS](https://astro.build/).

### Requerimientos necesarios

Para contribuir al proyecto y mantener la calidad de la página, es importante contar con los siguientes conocimientos y herramientas:

- Dominio de HTML, CSS y JavaScript
- Familiaridad con React
- Conocimiento de Astro JS
- Experiencia en el uso de control de versiones con **Git**
- Competencia en el uso de la terminal

Con estos requisitos, podrás participar activamente en el desarrollo y la mejora continua de nuestro proyecto.

## Cómo empezar

Para comenzar a contribuir, primero necesitarás clonar el repositorio del proyecto. Puedes hacerlo ejecutando el siguiente comando en la terminal (se requiere tener Git instalado):

```bash
git clone git@github.com:UnCompa/Proyecto-UEFP.git

```

Con esto se clonara el repositorio para empezar a trabar en ella.

### Estructura del proyecto

La estructura actual del proyecto es el siguiente:

```txt
Proyecto-UEFP/
│
├── public/
│   ├── imgs
│   └── videos
│   └── svg
│
├── src/
│   ├── components/
│   ├── i18n/
│   ├── icons/
│   ├── layouts/
│   ├── lib/
│   ├── meta/
│   ├── pages/
│   ├── schema/
│   ├── store/
│   └── utils
│
└── .gitignore
└── README.md
└── tailwind.config.mjs
└── package.json
└── package-lock.json
└── astro.config.mjs
└── tsconfig.json
```

#### Carpeta public

Esta carpeta contiene recursos estáticos utilizados en la página web, como imágenes y videos.

##### Imgs

Almacena imágenes principales utilizadas en la web.

##### Videos

Carpeta que almacena videos utilizados en la web.

##### Svg

Se guardan imágenes de tipo SVG utilizadas en la web.

#### Carpeta src

Esta carpeta contiene el código fuente de la aplicación, organizado en diferentes subdirectorios:

##### Components
Contiene componentes reutilizables utilizados en diferentes partes de la página.

##### i18n
Almacena las traducciones para español e inglés de la página.

##### Icons

Un componente de React que exporta iconos en formato SVG para su uso en la página.

##### Layouts

Aquí se encuentran las plantillas que proporcionan una base reutilizable para diferentes casos de uso.

##### Lib

Contiene bibliotecas útiles, como la biblioteca para utilizar Supabase

##### Meta

Contiene componentes con información necesaria para la indexación de la página.

##### Pages
La carpeta más importante, que contiene las rutas a las diferentes páginas de la web.
##### Schema

Carpeta para definir esquemas de datos.

##### Store

Almacena archivos relacionados con el manejo del estado de la aplicación.

##### Utils

Esta carpeta contiene utilidades varias utilizadas en el proyecto.
